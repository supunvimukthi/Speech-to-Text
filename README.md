# Speech-to-Text 

Electron app to run on windows to convert speech to text through google cloud api. This app allows user to change system volume of speaker and mic through [win-audio](https://www.npmjs.com/package/win-audio) package

## How to run the App 
 
- clone this git repo. 
- cd into Speech-to-Text. 
- run *'npm install'*. 
- run *'npm start'*
